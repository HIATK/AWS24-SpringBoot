select * from board
where (title like concat('%','1','%') or content like concat('%','1','%'))
order by bno desc;

이 SQL 쿼리는 board 테이블에서 특정 조건을 만족하는 레코드를 선택하고,
그 결과를 bno 필드를 기준으로 내림차순 정렬하여 반환합니다.
  조건은 title 필드가 '1'을 포함하거나 content 필드가 '1'을 포함하는 레코드를 선택합니다.
   like 연산자와 concat 함수를 사용하여 이를 수행합니다.
 concat('%', '1', '%')는 '1'이 어디에 위치하든 상관없이 해당 문자열을 포함하는 레코드를 찾습니다.
 정렬에 대한 내용이다

 where은 조건문이다 false 면 아무것도 없고 true면 결과가 나온다.
 불린타입으로 조건을 보고 있는것이고
 가로로 묶는 상황이 온다면 참 거짓으로 볼수 있는 상황이 만들어 진다.
 참이냐 거짓이냐를 보는 내용이다 불린 형태의 해당 결과를 넘겨 줄수 있으면 된다
 내용이 있으면 틈루
 없으면 폴스다
